# Page: 404 Not Found

## Route

`/404`

## Rendering

Static (pre-rendered at build time)

## Purpose

Friendly error page when a visitor hits a broken or non-existent URL.

## User Stories

- As a visitor who followed a broken link, I want to understand the page doesn't exist and find my way back.
- As a visitor with an old bookmark from the previous site, I want to be guided to the equivalent content.

## Sections

### 1. Error Message

- Heading: "Page Not Found" or something playful (e.g., "This bear has wandered off...")
- Brief message explaining the page doesn't exist
- Copy TBD — should match the site's personality

### 2. Navigation Suggestions

- Link to home page
- Link to bears section
- Link to products
- Optionally: a search suggestion or popular pages

### 3. Optional: Featured Bear

- Show a random bear image to keep the page on-brand
- "While you're here, meet {bear name}!" with link to profile

## Content Collections Used

- `bears` — optional random bear for personality

## Components Used

- Bear card (optional)

Uses `page.layout.astro` (or `base.layout.astro` directly for a more custom layout)

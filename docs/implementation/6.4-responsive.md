# 6.4 Responsive Design Pass

**Phase:** 6 — Gallery & Polish

**Goal:** All pages tested and polished at mobile, tablet, and desktop breakpoints across all 4 themes.

## User Stories

- As a mobile visitor, I want the site to work perfectly on my phone.
- As a tablet visitor, I want a comfortable browsing experience.

## Tasks

- [ ] Test all pages at mobile breakpoint (320px-640px) **in all 4 themes**
- [ ] Test all pages at tablet breakpoint (768px-1024px) **in all 4 themes**
- [ ] Test all pages at desktop breakpoint (1024px+) **in all 4 themes**
- [ ] Verify hamburger menu works correctly in all themes
- [ ] Verify card grids are responsive
- [ ] Verify map is usable on mobile
- [ ] Verify filter tabs scroll/wrap appropriately on mobile
- [ ] Test all 4 themes at all breakpoints

## Depends On

All page phases (1-4)

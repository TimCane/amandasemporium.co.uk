# 1.3 Layout & Navigation

**Phase:** 1 — Project Foundation

**Goal:** Base and page layouts, navbar, footer with theme switcher, view transitions, cookie banner.

## User Stories

- As a visitor, I want a consistent navbar on every page so I can navigate the site easily.
- As a visitor, I want the navbar to collapse into a hamburger menu on mobile.
- As a visitor, I want a footer with a subtle theme switcher.
- As a visitor, I want smooth page transitions when navigating between pages.
- As a visitor, I want the navbar and footer to look completely different in each theme.

## Tasks

- [ ] Create `base.layout.astro` (HTML shell, head, fonts, theme loading, view transitions, cookie banner, skip link)
- [ ] Create `page.layout.astro` (extends base, adds navbar + footer + page-wrapper + page-content divs)
- [ ] Create navbar component with sufficient HTML hooks for all 4 themes (wrappers, decoration divs)
- [ ] Create footer component with theme switcher and HTML hooks
- [ ] Create text logo component
- [ ] Create cookie banner component
- [ ] Add `<ViewTransitions />` to base layout with per-theme transition animations
- [ ] Add `transition:persist` to navbar and footer
- [ ] **Style navbar in all 4 themes** (Boutique: elegant thin bar, Whimsical: rounded soft, Modern: clean uppercase, Fun: chunky colourful)
- [ ] **Style footer in all 4 themes**
- [ ] **Style page layout wrappers in all 4 themes** (different max-widths, section backgrounds)

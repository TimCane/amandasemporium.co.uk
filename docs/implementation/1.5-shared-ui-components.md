# 1.5 Shared UI Components

**Phase:** 1 — Project Foundation

**Goal:** Reusable UI components (card, pagination, filter tabs) used across multiple pages.

## User Stories

- As a developer, I want reusable UI components so I can build pages consistently and quickly.

## Tasks

- [ ] Create base card component (`src/components/ui/card.astro`)
- [ ] Create pagination component (`src/components/ui/pagination.astro`)
- [ ] Create filter tabs component (`src/components/ui/filter-tabs.astro`)
- [ ] Create `src/libs/consts.ts` with site title and other constants

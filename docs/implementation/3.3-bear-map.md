# 3.3 Bear Map

**Phase:** 3 — Bears Section

**Goal:** Interactive Leaflet map showing all bear locations with clustering and journey paths.

## User Stories

- As a visitor, I want to see all bear locations on an interactive map.
- As a visitor, I want to click a marker to see which bear it is and visit their profile.
- As a visitor, I want to see journey paths between rescue and rehome locations.

# 3.2 Bear Profile Pages (Static)

**Phase:** 3 — Bears Section

**Goal:** Individual bear profile pages with full story, photos, map, and related bears.

## User Stories

- As a visitor, I want to see a bear's full story — photo, narrative, rescue and rehoming journey.
- As a visitor, I want to browse additional photos of the bear if available.
- As a visitor, I want to see the bear's journey on a small map.
- As a visitor, I want to discover related bears from the same location, event, or species.
- As a visitor, I want to navigate back to the bears listing easily.

- [X] Build `/bears/[slug].astro` with `getStaticPaths` and HTML hooks for theme layouts

# 6.3 Accessibility Audit

**Phase:** 6 — Gallery & Polish

**Goal:** WCAG AA compliance across all pages and all 4 themes.

## User Stories

- As a visitor using a screen reader, I want to navigate the site meaningfully.
- As a visitor who can't use a mouse, I want to use the site via keyboard only.
- As a visitor with low vision, I want sufficient colour contrast in all themes.

## Tasks

- [ ] Add skip-to-content link
- [ ] Verify all images have meaningful alt text
- [ ] Verify all interactive elements are keyboard accessible
- [ ] Verify focus indicators are visible in all themes
- [ ] Check colour contrast ratios for all 4 themes (WCAG AA minimum)
- [ ] Verify ARIA attributes on dynamic components (navbar, filter tabs, pagination, map)
- [ ] Test with screen reader
- [ ] Verify semantic HTML (landmarks, headings hierarchy)

## Depends On

All page phases (1-4)

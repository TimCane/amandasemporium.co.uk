# 4.3 Contact Page

**Phase:** 4 — Static Pages

**Goal:** Contact page with form, FAQ section, and next event callout.

## User Stories

- As a visitor, I want to see Amanda's email address so I can contact her directly.
- As a visitor, I want to send a message via a contact form without leaving the site.
- As a visitor, I want to see confirmation that my message was sent.
- As a visitor, I want to find answers to common questions without needing to email.
- As a visitor, I want to know about the next event so I can visit in person.

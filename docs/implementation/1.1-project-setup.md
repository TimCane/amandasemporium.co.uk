# 1.1 Project Setup

**Phase:** 1 — Project Foundation

**Goal:** A fresh Astro 5 project with SCSS, TypeScript, and Node adapter configured.

## User Stories

- As a developer, I want a fresh Astro 5 project with TypeScript so I have a solid starting point.
- As a developer, I want SCSS configured with CSS custom properties so I can build theme-aware components.
- As a developer, I want the Node adapter configured in hybrid mode so I can opt routes into SSR.

## Tasks

- [x] Initialise Astro 5 project (`npm create astro@latest`)
- [x] Install `sass` dependency for SCSS compilation
- [x] Create SCSS file structure (`_reset.scss`, `_accessibility.scss`, `_grid-areas.scss`, `_base.scss`, `_mixins.scss`, `global.scss`)
- [x] Install and configure `@astrojs/node` adapter (standalone, hybrid output)
- [x] Install and configure `@astrojs/sitemap`
- [x] Set up TypeScript config with `@` path alias
- [x] Create `.dockerignore`
- [x] Set up project directory structure (`components/`, `layouts/`, `libs/`, `styles/`, `content/`)

## Depends On

None — this is the first task.

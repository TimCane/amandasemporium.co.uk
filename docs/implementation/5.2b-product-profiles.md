# 5.2b Product Profile Pages (Static)

**Phase:** 5 — Products & Events

**Goal:** Individual product profile pages with full details, photos, how-it's-made, and related products.

## User Stories

- As a potential customer, I want to see a product's full details, photos, and how it's made.
- As a visitor, I want to appreciate the craftsmanship behind each product.
- As a visitor, I want to see other products in the same category.

## Tasks

- [ ] Build `/products/[slug].astro` with `getStaticPaths` and HTML hooks
- [ ] Create product profile structure (hero photo(s), name, price, narrative, how it's made, materials, decoration divs)
- [ ] Support multiple photos (theme decides display)
- [ ] Render markdown body as product story (no narrative section if body empty)
- [ ] Render `## How It's Made` section from body (if present)
- [ ] Display price, materials, category
- [ ] Implement "More in this category" related products section
- [ ] Add SEO: OG tags with product photo, JSON-LD Product schema
- [ ] **Style product profile in all 4 themes**
- [ ] Update product cards on listing page to link to `/products/[slug]`

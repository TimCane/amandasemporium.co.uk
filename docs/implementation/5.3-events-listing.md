# 5.3 Events Listing Page (SSR)

**Phase:** 5 — Products & Events

**Goal:** Events listing with upcoming/past toggle, year filter, and calendar integration.

## User Stories

- As a potential customer, I want to see upcoming events so I can plan to visit.
- As a visitor, I want to add an event to my calendar.
- As a visitor, I want to see past events to understand how established the business is.
- As a visitor, I want to filter past events by year.

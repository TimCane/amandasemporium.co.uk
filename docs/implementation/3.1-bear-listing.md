# 3.1 Bear Listing Page (SSR)

**Phase:** 3 — Bears Section

**Goal:** Paginated, filterable bear listing with SSR, alphabet nav, and status filters.

## User Stories

- As a visitor, I want to browse all rescued bears in a paginated grid.
- As a visitor, I want to filter bears by status (all / looking for home / found their home).
- As a visitor, I want to filter bears by the first letter of their name.
- As a visitor, I want to share a filtered URL with someone (e.g., "all bears starting with B").
- As a visitor, I want filters to feel smooth thanks to view transitions.

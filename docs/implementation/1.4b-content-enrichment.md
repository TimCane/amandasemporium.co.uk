## Notes

Content enrichment (writing taglines, narratives, etc.) is an **ongoing effort** separate from the build. Pages should handle empty taglines/bodies gracefully:
- Empty tagline: component auto-generates from structured data ("A {species} by {brand} from {location}")
- Empty body: profile page shows structured data only, no narrative section
- Missing `## How It's Made`: section hidden on product profile
- Missing `## Recap`: section hidden on event profile

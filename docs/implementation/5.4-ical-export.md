# 5.4 iCal Export

**Phase:** 5 — Products & Events

**Goal:** API endpoint to download events as iCalendar files.

## User Stories

- As a visitor, I want to download an event to my calendar app so I don't forget.

## Tasks

- [ ] Build `/events/calendar/[...slug].ics.ts` API endpoint
- [ ] Generate valid iCalendar format using `ics` library
- [ ] Include event name, date, time, location, tagline (or body excerpt as description), website
- [ ] Set correct Content-Type and Content-Disposition headers
- [ ] Add "Add to calendar" button to event cards (upcoming only)

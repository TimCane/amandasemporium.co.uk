# 4.1 Home Page

**Phase:** 4 — Static Pages

**Goal:** Feature-rich home page with hero, stats, upcoming events, featured products, bear of the month, and activity feed.

## User Stories

- As a first-time visitor, I want to immediately understand what Amanda's Emporium is.
- As a returning visitor, I want to see the next upcoming event quickly.
- As a visitor, I want to see featured products and the bear of the month.
- As a visitor, I want to see impressive stats that show the scale and history of the business.
- As a visitor, I want to see recent activity so the site feels alive and current.
- As a returning visitor, I want to see a different featured bear each month.
- As a visitor, I want clear paths to explore bears, products, and events.

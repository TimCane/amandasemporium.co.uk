# 4.1 Home Page

**Phase:** 4 — Static Pages

**Goal:** Feature-rich home page with hero, stats, upcoming events, featured products, bear of the month, and activity feed.

## User Stories

- As a first-time visitor, I want to immediately understand what Amanda's Emporium is.
- As a returning visitor, I want to see the next upcoming event quickly.
- As a visitor, I want to see featured products and the bear of the month.
- As a visitor, I want to see impressive stats that show the scale and history of the business.
- As a visitor, I want to see recent activity so the site feels alive and current.
- As a returning visitor, I want to see a different featured bear each month.
- As a visitor, I want clear paths to explore bears, products, and events.

## Tasks

- [ ] Build `/index.astro` with section landmark classes and decoration divs per section
- [ ] Create hero section with HTML hooks (image wrapper, overlay, decoration)
- [ ] Create about teaser section (short intro, link to /about)
- [ ] Create stats section (bears rescued, homes found, events attended, years active — computed from collections)
- [ ] Create upcoming events section (next 2-3 events from collection)
- [ ] Create featured products section (3-4 product cards — placeholder until Phase 4)
- [ ] Create photo strip section (horizontal band of gallery images)
- [ ] Create Bear of the Month section (seeded-random `YYYY-MM`, only bears without `rehomed`)

- [ ] Add SEO: OG tags, JSON-LD WebSite schema
- [ ] **Style home page in all 4 themes** (Boutique: centred editorial with gold dividers; Whimsical: wavy sections with soft cards; Modern: full-bleed hero with stats bar and alternating dark/light sections; Fun: colourful section backgrounds with chunky cards)

Each set of three bookmarks is cut and sewn by hand using coordinating quilting cottons, with a ribbon loop so they stay put in the page. Because fabrics change with each batch, no two sets are identical — you're getting something genuinely one-of-a-kind.

## How It's Made

The fabric is chosen in groups of three to make sure the prints complement each other. Each bookmark is interfaced for stiffness, top-stitched on all four sides, and finished with a folded ribbon loop at the top. The whole set is packaged together with a small card explaining the care instructions.

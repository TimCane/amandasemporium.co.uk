Each bookmark is cut and sewn by hand using coordinating quilting cottons, with a ribbon loop or tassel so they stay put in the page. Because fabrics change with each batch, no two are identical — you're getting something genuinely one-of-a-kind.

## How It's Made

The fabric is selectively chosen to make sure the prints complement each other. Each bookmark is interfaced for stiffness, top-stitched on all four sides, and finished with a folded ribbon loop at the top.
